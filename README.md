# DistroBOX v1.1

**DistroBOX** ek lightweight Bash script hai jo Termux (Android) par alag-alag Linux distributions ko aasan tarike se install karne ke liye banaya gaya hai. Yeh script `proot` aur `AnLinux` resources ka istemal karke aapke Android device par Linux environment set up karta hai.

---

## 🚀 Features

* **One-Click Installation:** Lambe commands likhne ki jarurat nahi, bas option select karein aur install karein.
* **Multiple Distros:** Ek hi jagah par kaafi saare popular Linux OS ka support:
    * Kali Linux (Nethunter)
    * Parrot OS
    * Ubuntu
    * Debian
    * Fedora
    * CentOS Stream
    * openSUSE Leap
    * blackbox"
    * black arch
    * arch linux
    * alpine
    * Void linux
    * open SUSE Tumbleweed

* **Termux Friendly:** Isse khasskar Termux users ke liye optimize kiya gaya hai (`pkg` manager ka use karke).

---

## 🛠️ Requirements

Is script ko chalane ke liye aapko **Termux** app ki jarurat hogi. Agar install nahi hai, toh aap isse [F-Droid](https://f-droid.org/) se download kar sakte hain.
